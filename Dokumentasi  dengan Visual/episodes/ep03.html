Episode 03: Cara Kerja Program

Alur Eksekusi Python

Berbeda dengan bahasa C yang di-compile ke bahasa mesin, Python memiliki alur unik:

  1. Source Code (.py): Kode yang kita ketik.
  2. Bytecode (.pyc): Kode perantara yang lebih "dimengerti" oleh mesin Python.
  3. PVM (Python Virtual Machine): Komponen yang menjalankan bytecode tersebut ke sistem operasi.

Cara Cek Bytecode:

Coba ketik ini di terminal untuk melihat keajaiban bytecode:

python -m py_compile main.py

Setelah itu, cek folder __pycache__ di direktori kamu.
