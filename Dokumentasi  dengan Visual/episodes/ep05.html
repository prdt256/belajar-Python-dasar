🐍 Episode 05: Tipe Data Dasar & Khusus

Mengenal Tipe Data

Tipe data adalah jenis-jenis data yang bisa dimasukkan ke dalam variabel dan diproses oleh Python. Untuk mengecek jenis tipe data suatu variabel, kita bisa menggunakan fungsi bawaan type().

Tabel Tipe Data Dasar

Tipe Data Class Penjelasan Contoh
Integer int Angka bulat tanpa desimal 10, -5
Float float Angka pecahan/desimal 3.14, 1.5
String str Kumpulan karakter/teks "Ucup", "10"
Boolean bool Nilai logika (Benar/Salah) True, False

Tipe Data Khusus & C-Types

Selain tipe data dasar, Python juga mendukung bilangan matematika complex (seperti 5 + 6j) dan tipe data murni dari bahasa C dengan memanfaatkan modul ctypes untuk kebutuhan komputasi performa tinggi.
