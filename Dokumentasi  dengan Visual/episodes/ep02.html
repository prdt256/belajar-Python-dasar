Episode 02: Instalasi & Persiapan

Sebelum mulai ngoding, kita perlu menyiapkan "senjata" utama kita.

1. Python Interpreter

Download di python.org. Pastikan centang "Add Python to PATH" saat instalasi.

2. Code Editor (VS Code)

Gunakan Visual Studio Code untuk menulis kode yang lebih efisien dengan bantuan extension Python.

3. Terminal

Tempat kita mengeksekusi perintah. Gunakan python nama_file.py untuk menjalankan program.
