🔄 Episode 06: Casting Tipe Data

Apa itu Casting Tipe Data?

Casting adalah mekanisme untuk mengubah jenis data dari satu tipe ke tipe lainnya. Kita menggunakan fungsi dasar seperti int(), float(), str(), dan bool().

💡 Aturan Logika Boolean (bool)

  • False: Hanya terjadi jika nilainya adalah angka 0, desimal 0.0, atau string kosong "".
  • True: Terjadi pada angka selain 0 (termasuk angka negatif) dan string yang memiliki isi karakter sekecil apa pun (termasuk spasi).

⚠️ Potensi Error (ValueError) pada String

Berhati-hatilah saat melakukan casting dari String ke tipe angka:

  • int("10") ➡️ Aman (menjadi 10).
  • int("10.5") ➡️ ERROR! String desimal tidak bisa langsung dipaksa jadi Integer bulat.
  • int("Ucup") ➡️ ERROR! Teks huruf tidak bisa diubah menjadi angka.
