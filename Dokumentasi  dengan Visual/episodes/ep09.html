⚖️ Episode 09: Operasi Komparasi

Hasil Akhir Komparasi

Operasi komparasi selalu menghasilkan tipe data Boolean, yaitu True atau False.

Operator Arti Contoh Kode Hasil
== Sama Dengan 4 == 2 False
!= Tidak Sama Dengan 4 != 2 True
is Identitas Memori Sama x is y True / False

⚠️ Perbedaan == dengan is

Operator == membandingkan NILAI-nya, sedangkan operator is membandingkan ALAMAT MEMORI-nya (apakah mereka objek yang sama persis atau bukan).
