🧮 Episode 08: Operasi Aritmatika

Tiga Operator Unik Python

Selain operator standar sekolah, Python punya 3 operator andalan yang sangat berguna:

  • ** Eksponen: Digunakan untuk perpangkatan. Contoh: 2 ** 3 = 8.
  • % Modulus: Mencari sisa dari hasil pembagian. Sangat berguna untuk menentukan bilangan ganjil/genap.
  • // Floor Division: Pembagian yang hasilnya langsung dibulatkan ke bawah (menghilangkan angka di belakang koma).

KABATAKU Matematika Python

Jika dalam satu baris kode terdapat banyak operator, Python akan mengeksekusinya berdasarkan urutan kasta/prioritas berikut:

  1. Operasi di dalam kurung ( )
  2. Perpangkatan / Eksponen **
  3. Perkalian, Pembagian, Modulus, Floor Division *, /, %, //
  4. Penjumlahan dan Pengurangan +, -

🌡️ Proyek Kecil: Konversi Suhu

Menggabungkan Input, Casting, dan Aritmatika:

  • Celcius ke Reamur: (4/5) * C
  • Celcius ke Fahrenheit: (9/5) * C + 32
  • Celcius ke Kelvin: C + 273
