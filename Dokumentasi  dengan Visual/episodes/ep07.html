⌨️ Episode 07: Mengambil Input Data dari User

Fungsi Utama: input()

Fungsi ini akan menghentikan jalannya program sejenak dan menunggu user mengetikkan sesuatu di terminal lalu menekan tombol Enter.

⚠️ Hukum Mutlak input()

Apapun yang diketik oleh user, baik itu nama, angka bulat, angka desimal, bahkan kata "True/False", Python akan selalu membacanya sebagai STRING (Teks).

User Ketik: 10 ➡️ fungsi input() ➡️ Hasil di Python: "10" (String)

Cara Mengambil Input Angka (Kombinasi Casting)

Jika ingin melakukan operasi aritmatika pada data inputan, kita wajib membungkus fungsi input dengan tipe data yang diinginkan:

# Mengambil input angka bulat
usia = int(input("Masukkan usia: "))

# Mengambil input angka desimal
tinggi = float(input("Masukkan tinggi badan (m): "))
